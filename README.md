# FindWeb
 Projeto de estudo desenvolvido utilizando apis, javascript e tmp bootstrap

 ### Funções:

 - Consulta de IP
 - Consulta de CEP
 - Consulta de CNPJ
 - Consulta de DDD
 - Consulta de BIN
 - Consulta de Banco
 - Consulta de informação sobre Bitcoin
 - Gerador de pessoas fictícias
 - Gerador de CPF
 - Gerador de CNPJ
 - Gerador de CNS
 - Gerador de senhas
 - Criptografar em MD5
